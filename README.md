# ✂️ Stone Paper Scissors Game

A fun **Stone Paper Scissors** game where you play 
against the computer, built using **HTML, CSS & JavaScript**.

## 🚀 Features
- Player vs Computer mode 
- Random computer move generation
- Score tracking system
- Win, Lose & Draw detection
- Responsive & interactive UI

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## 🎯 How to Play
1. Choose Stone, Paper or Scissors
2. Computer randomly picks its move
3. Winner is decided instantly!
4. Score updates after every round
